# Aung Academy Language School Website

## Description

Aung Academy Language School is a freelance web project developed to showcase language courses and provide information about the academy.
The website includes a public frontend for visitors and an admin dashboard for managing courses and website content.

## Features

* Home page to introduce the language school
* About page with academy information
* Courses page to display available language courses
* Admin dashboard to manage courses
* Responsive modern UI

## Technologies Used

* Laravel (Backend Framework)
* Vue.js (Frontend Framework)
* Tailwind CSS (Styling)

## Installation

1. Clone the repository

git clone https://github.com/minthukha-coder/aals-school.git

2. Install backend dependencies

composer install
composer update

3. Install frontend dependencies

npm install

4. Run the development server

php artisan serve

5. Run frontend

npm run dev

## Project Structure

* Admin Dashboard for managing courses
* Public pages: Home, About, Courses
